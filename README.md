# heart-disease-predictor

This project uses logistic regression to predict the presence of heart disease from patient clinical data.

# Tools
- Python
- pandas, NumPy
- scikit-learn
- Google Colab

# What it does
- Cleans and preprocesses raw data
- Encodes categorical features
- Handles missing values
- Trains a logistic regression model
- Evaluates performance on train and test sets
